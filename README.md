# Scivity
Scivity is a mod for minecraft 1.12.2 adding different blocks and items with a technological and industrial theme.

---

## Installation Instructions
1. Download the source from [here](https://bitbucket.org/BenjaminHeath238/scivity/get/d5b0bae09732.zip) or via git using `git clone https://BenjaminHeath238@bitbucket.org/BenjaminHeath238/scivity.git`
2. Unzip the downloaded zip file and Enter the folder containing the `build.gradle` file
3. Open a shell/terminal, in the same directory, and run `./gradlew build` for Linux or `gradlew build` for Windows
4. Wait for the compilation process to finish then collect the generated file from `build/libs/scivity-X.X.jar` or `build\libs\scivity-X.X.jar` for Linux or Window systems respectively
5. Add to Minecraft 1.12.2 with forge instance and play
